 什么是thinkgl？
  
  
     thinkgl 是由thinkia写的一个webgl库

  为什么要写thinkgl?


       目前已有three.js,babylon.js等主流webgl库；国内也有蛮好的库比如claygl,G3D,Hilo3d,他们的作者都是很棒很有趣的人；我
       
       为什么要写thinkgl呢？一方面希望能够更深刻的理解webgl提高自己的水平;另一方面希望留下点有用的东西：从网上学了那么多
       
       知识总得归还点东西吧？

       我又有什么东西可以归还呢？对webgl的理解？好像也并不深刻…… 我会告诉你，我看到的webgl是一头大象，而且认认真真摸了
       
       摸大象。嗯，大象长得像一个大蒲扇。如果你并不觉得可笑，还有耐心看下去，我会觉得非常荣幸，谢谢你；接下来我会分享一
       
       下摸象的过程和一些蒲扇的细节；
       
  和其他的webgl库有什么区别？
      
      
       并没有本质区别。都是对webgl API的封装只是形式和方法不一样；比如：主流框架都是喜欢用scene,camera这样的概念来描绘一
       
       个三维场景； 而在thinkgl里面，你需要想象自己来到了一个叫做iaWorld的三维异世界，用眼睛观察着这个世界的一切；来到这
       
       个世界要做的第一件事情就是张开眼睛 action.eyes.openEyes( fov ,near ,far ,aspect) ,如果要改变观察方式你需要眨一眨
       
       眼睛action.eyes.blink( fov ,near ,far ,aspect); 它的本质其实还是改变投影矩阵，只是在thinkgl里面换了一种说法而已;
